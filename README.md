# 🧭 SmartNav – AI-Powered Travel Companion

SmartNav is a full-stack mobile application that helps travelers explore destinations, discover nearby attractions, receive personalized AI travel assistance, monitor weather conditions, and stay safe while traveling.

The app combines artificial intelligence, location-based services, weather intelligence, and emergency features into a single travel platform designed to enhance the overall travel experience.

---

## ✨ Features

### 🤖 AI Travel Assistant
- Personalized travel recommendations
- Destination insights and suggestions
- Context-aware travel guidance
- Interactive AI-powered assistance

### 📍 Smart Destination Discovery
- Find nearby attractions and landmarks
- Discover restaurants, cafes, and hotels
- Explore points of interest based on location
- Real-time place recommendations

### 🌤 Weather Intelligence

- Travel condition monitoring
- Weather-based travel recommendations

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/sebinjoseph005/smartnav.git
cd smartnav
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
SUPABASE_URL=YOUR_SUPABASE_URL
SUPABASE_ANON_KEY=YOUR_SUPABASE_KEY
GROQ_API_KEY=YOUR_GROQ_API_KEY
FOURSQUARE_API_KEY=YOUR_FOURSQUARE_API_KEY
```

### Run Backend

```bash
npm run server
```

### Run Mobile Application

```bash
npx expo start
```
